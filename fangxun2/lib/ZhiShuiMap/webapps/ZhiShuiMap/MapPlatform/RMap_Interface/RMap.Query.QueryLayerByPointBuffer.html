queryLayerByPointBuffer (lng, lat, radius, layerName, option)
点缓冲查询
参数 类型 描述
lng Number 待缓冲点经度
lat Number 待缓冲点纬度
radius Number 缓冲半径（单位：米）
layerName String 图层名称，从图层资源列表中获取
option Object 可选参数
        queryWhere String 属性查询条件。示例：NAME=’张三’ and AGE>20
        geoQueryType String 几何查询条件类型（相交：intersects，包含：within）
        styleIn pointStyle| lineStyle| polygonStyle 几何查询条件样式
        styleOut pointStyle| lineStyle| polygonStyle 查询结果样式
        onClick Function 当点击结果时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}
        onResult Function 查询完成后的回调函数，详细参数见回调参数表示例：function(event) {…}
onClick回调函数参数
参数 类型 描述
event Object 返回参数
        result Object 属性列表
onResult回调函数参数
参数 类型 描述
event Object 返回参数
        result Object 查询结果，对应返回字段以及内容
点样式
参数 类型 描述
pointStyle Object 可选参数
        pic String 自义图片的路径
        anchor Array 图标偏移。默认不传，偏移量为图标宽、高的一半（像素值）。[x,y]：x大，往左偏移；y大，往上偏移；示例[10,10]。
线样式
参数 类型 描述
lineStyle Object 可选参数
        width Number 线条的宽度
        color String 颜色值，示例：'#FF000'、 'red'。
        opacity Number 透明度，0-1。
面样式
参数 类型 描述
polygonStyle Object 可选参数
        outLineWidth Number 外部线条样式，线条的宽度
        outLineColor String 外部线条样式，线条颜色值，示例：'#FF000'、 'red'。
        outLineOpacity Number 外部线条样式，线条透明度，0-1。
        color String 颜色值，示例：'#FF000'、 'red'。
        opacity Number 透明度，0-1。
