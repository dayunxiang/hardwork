center (lng, lat, option)
设置地图居中显示，地图会平移到指定的位置
参数 类型 描述
lng Number 经度
lat Number 纬度
option Object 可选参数（暂留）
zoom (level)
设置地图放大、缩小，地图会放大、缩小到指定的层级
参数 类型 描述
level Number 地图层级
centerZoom (lng, lat, level, option)
地图平移、缩放，地图会平移缩放到指定的位置和层级
参数 类型 描述
lng Number 经度
lat Number 纬度
level Number 地图层级
option Object 可选参数（暂留）
extent (minX, minY, maxX, maxY, option)
地图缩放到指定范围
参数 类型 描述
minX Number 最小经度（左下角）
minY Number 最小纬度（左下角）
maxX Number 最大经度（右上角）
maxY Number 最大纬度（右上角）
option Object 可选参数（暂留）
extentByData (pointList, option)
地图缩放到指定数据范围
参数 类型 描述
pointList Array.<[lng,lat]> 点位数据集合
option Object 可选参数（暂留）
