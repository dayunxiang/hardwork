locationToAddress (lng, lat, onResult, option)
坐标转地址
参数 类型 描述
lng Number 经度
lat Number 纬度
onResult Function 转换成功回调函数，回调时会将转换出的所有信息作为参数传回。示例：function(event){…}
option
onResult回调函数参数
参数 类型 描述
event Object 返回参数
        address Object 在线版：详细地址，省＋市＋区/县＋乡/镇/街道＋临近POI数据 。
离线版：详细地址，省＋市＋区/县＋乡/镇/街道＋门牌号；临近路口位置
        province Object 省
        city Object 市
        district Object 区/县
        township Object 乡/镇/街道
