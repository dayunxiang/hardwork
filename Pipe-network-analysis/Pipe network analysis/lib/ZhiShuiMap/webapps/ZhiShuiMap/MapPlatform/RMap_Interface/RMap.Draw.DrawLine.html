drawLine (callback, option)
绘制折线
参数 类型 描述
callback Function 绘制完成后的回调函数，详细参数见回调参数表。示例：function(event) {…}
option Object 可选参数
        id String 绘制图形的id，在删除时使用
        width Number 线条的宽度
        color String 颜色值，示例：'#FF000'、 'red'。
        opacity Number 透明度，0-1。
回调事件参数
参数 类型 描述
event Object 返回参数
        id String 绘制图形的id，在删除时使用
        minPoint [lng,lat] 矩形的左下角（最小）坐标
        maxPoint [lng,lat] 矩形的右上角（最大）坐标
        extent Array.<Number> 圆形外界矩形的左下角（最小）和右上角（最大）坐标。
        pointList Array.<[lng,lat]> 组成折线的点
removeDrawLine (drawIds, option)
移除绘制的图形
参数 类型 描述
drawIds Array.<draw.id>|draw.id 地图上已绘制图形的id的数组集合。如果不传则移除所有绘制图形
option Object 可选参数（暂留）
