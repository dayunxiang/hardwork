addPoint (lng, lat, option)
在地图上添加点位
参数 类型 描述
lng Number 经度
lat Number 纬度
option Object 可选参数
        id String 标记id，id不可重复，在点位高亮、移除等操作时会用到。注意：在不指定id的值时，系统会自动生成id。但在多个对象同时指定为一个id时。地图上只会加载第一个点。
        pic String 自定义图标的路径
        anchor Array 图标偏移。默认值为图片宽、高的一半（像素值）。参数格式：[x,y] x大，往左偏移；y大，往上偏移。示例：一张50*50大小的图片，左上角的坐标是[0,0],右下角的坐标是[50*50],在不传此参数时系统默认会设置为[25,25]。
        tooltip HTMLDivElement 鼠标移入时显示提示信息，参数内容为html的div页面
        tooltipDirection String 提示信息窗口的默认显示方向，可选值有：right, left, top, bottom, center, auto。默认为auto
        data Object 用于自定义传参
        onClick Function 当点击标记时执行的回调函数，回传的参数参见onClick回调函数参数表。示例：function(event){…}
        onMouseOver Function 当鼠标移入标记时执行的回调函数，回传的参数参见onMouseOver回调函数参数表。示例：function(event){…}
        onMouseOut Function 当鼠标移出标记时执行的回调函数，回传的参数参见onMouseOut回调函数参数表。示例：function(event){…}
        onTooltipOpen Function 当显示提示信息窗口时执行的回调函数，回传的参数参见onTooltipOpen回调函数参数表。示例：function(event){…}
onClick回调函数参数
参数 类型 描述
event Object 返回参数
        lng Number 经度
        lat Number 纬度
        data Object 自定义的传参
        id String 标记id
onMouseOver回调函数参数
参数 类型 描述
event Object 返回参数
        lng Number 经度
        lat Number 纬度
        data Object 自定义的传参
        id String 标记id
onMouseOut回调函数参数
参数 类型 描述
event Object 返回参数
        lng Number 经度
        lat Number 纬度
        data Object 自定义的传参
        id String 标记id
onTooltipOpen回调函数参数
参数 类型 描述
event Object 返回参数
        lng Number 经度
        lat Number 纬度
        data Object 自定义的传参
        id String 标记id
removePoint (pointId, option)
移除地图上的点位，如果不传参数则移除地图上的所有点位
参数 类型 描述
pointId Array.<point.id>|point.id 地图上已添加点位的id的数组集合。如果不传则移除所有点位
option Object 可选参数（暂留）
